Scenario

This dynamic SolverVBD cable example starts from a twist-free hanging arc, keeps both endpoint positions fixed, and gradually counter-twists the endpoint material frames. Past the buckling threshold the centerline leaves the initial plane and folds into a plectoneme held open by hard-history self-contact.

The Michell and twisted-ring pages cover closed-loop buckling behavior. This page is the open-cable dynamic contact case.

Strain Measure Comparison

The same plectoneme scenario run with two strain measures. Körner/Audoly (2021) uses a quaternion rotation-vector bend measure bounded at ±2; DER uses the discrete elastic rod curvature binormal (unbounded) with Bishop-frame parallel transport for twist.

The Körner implementation is correct per the paper (κ = 2 Im(qparent−1 qchild), bounded by the principal quaternion branch). The wider coil geometry is a property of the bounded bend measure — not an implementation defect. DER's unbounded curvature binormal raises the energy cost of tight bends, producing a geometrically tighter plectoneme.
