VBD Contact Friction Ramp

Box friction validation with stick/deadzone helpers disabled, covering static ramp classification, dynamic ramp sliding, and dynamic slide stop distance.

Modessoft / hard
Reference Iterations8
Stiffness1e4 / 1e7

Setup

Static Ramp starts boxes at rest on ramps and checks the Coulomb threshold: below atan(mu) they should stick, above it they should slide.

Dynamic Ramp uses the same ramp grid with downhill initial velocity. Dynamic Slide uses flat tracks and checks horizontal stop distance against analytic Coulomb sliding.

Result

Static Ramp. Hard contact with ke=1e7 gives strong static holding in this box test.
Dynamic Ramp. The same trend appears on ramp geometry; on-ramp error is used while boxes remain on the finite ramp.
