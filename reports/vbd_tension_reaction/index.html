What Is Reported

SolverVBD now populates State.body_parent_f when the state attribute is requested with ModelBuilder.request_state_attributes("body_parent_f"). Each entry stores the incoming parent-to-child joint wrench in world frame, referenced to the child body's center of mass.

This matches the convention used by Newton's other parent-force outputs. The values are VBD's applied reaction for the accepted iterative solve, so the validation checks force and torque balance against the simulated acceleration rather than assuming an exact continuous-time trajectory. The main plots validate default hard-mode structural joints. A soft-mode comparison using the same tuned stiffness is included as a separate test after the hard-mode results.
Hard structural slots accepted AL reaction, i.e. the post-update lambda-style load
Soft structural slots penalty reaction from the soft constraint force law
Drives, limits, damping not used in these tests; included by the readout when present

Budget And Stiffness

The baseline uses 4 VBD iterations, 4 substeps, and a frame timestep of dt = 1 / 60. That is 16 VBD iterations per rendered frame, with each solve using a substep timestep of 1 / 240 s.

Constraint mode Default hard structural joints
Baseline timestep 1 / 60 s
Baseline iterations / substeps 4 iterations, 4 substeps
Stiffness rigid_joint_linear_ke = 1.0e4, rigid_joint_angular_ke = 1.0e5, rigid_avbd_beta = 0.0
The baseline uses a tuned linear stiffness below the default so the report can compare reaction readout quality without hiding everything behind very stiff stabilization. Hard structural joints report the accepted AL reaction after the current residual has been folded into lambda; the soft comparison reports the corresponding penalty-mode reaction.

Analytic Balance

For a single child body with only gravity and the inbound joint reaction, Newton's law gives R = m a - m g. For the second body of the double pendulum the same expression applies. For the first body, the root joint also carries the child link load: R0 = m0 (a0 - g) + R1.

The torque check uses the joint moment arm from the child COM: tau = r_joint x R.

The video side plots use these balance equations as the reference. Their x-axis is time in seconds. Force residuals are plotted separately in N, and torque residuals are plotted separately in N m.

In the plot legends, reference means this analytic balance value computed from the simulated acceleration, for example Fx reference is the x-component of R = m a - m g.

Force component plots show Fx, Fy, and Fz separately. These pendula move in the x-z plane around revolute y-axes, so the component torque plots focus on Ty; the residual plots still use the full 3D torque norm.

Load Scale

The force errors are small compared with the loads carried by the joints. For scale, the single pendulum has a 1.000 kg body, so its static weight is 9.810 N. Each double-pendulum link is 0.512 kg, with a 5.023 N link weight and a 10.045 N root-subtree weight.

Results

Single Pendulum

The reported joint force tracks the balance expression over a dynamic swing. The maximum force residual is 0.010 N and the RMS force residual is 0.003 N, about 0.024% of the RMS analytic load.

Double Pendulum

With 4 iterations and 4 substeps, the double pendulum has a 0.121 N RMS force residual. The middle joint balances the second link directly, while the root joint balances the first link plus the child-link reaction. The root is the harder readout, at 0.172 N RMS force error against a 10.785 N RMS analytic load.

Soft-Mode Comparison

The same scenes were also run with the structural joint slots switched to soft penalty mode, using the same tuned stiffness and solve budget. This is a separate readout test, not a replacement for hard constraint validation: soft joints trade exact constraint enforcement for penalty compliance.

Budget Sweeps

The sweeps keep the frame timestep fixed at dt = 1 / 60. The iteration sweep varies iterations 2, 4, 8, 16 at 4 substeps. The substep sweep fixes 4 iterations and varies substeps 1, 2, 4, 8.

RMS Residual Trends

The double pendulum is the sensitive case. At 4 substeps, increasing iterations from 4 to 16 drops RMS force residual from 0.121 N to 0.003 N. At 4 iterations, increasing substeps from 1 to 8 drops RMS force residual from 0.943 N to 0.022 N.
