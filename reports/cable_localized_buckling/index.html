Verification Target

This example uses the DER localized-buckling parameter set: a naturally straight rod of length 9.29, bend modulus 1.345, twist modulus 0.789, 27 clamp turns, and 0.3 units of axial shortening. Root and tip are kinematic; the interior rod uses Newton's cable stretch, geometric bend, and transported-twist energy. The report shows the no-inertia quasistatic reproduction used for validation. The run is initialized with the requested 27-turn material-frame path, but final path-continuous twist unwrapping is not yet supported, so this page validates the localized centerline equilibrium rather than final 27-turn twist retention. Endpoint-quaternion driving is intentionally excluded from the validation path because it is branch-limited for this 27-turn case.

The analytic boundary-value report checks exact low-deformation bend and twist separately. This example stresses large twist, axial shortening, and bend/twist coupling in one open-rod setup.

Acceptance Criteria

Passes When

  • Orange centerline should leave the gray straight reference.
  • The buckle amplitude should localize instead of becoming a uniform sideways drift.
  • Blue clamp markers should remain fixed.

Watch For

  • rod stays nearly straight
  • tip clamp drifts away from the prescribed shortened position
  • segments visibly stretch or collapse

Validated Behavior

  • Newton's cable energy reproduces the localized-buckling equilibrium branch
  • large imposed twist couples into centerline buckling
  • the kinematic clamp position remains enforced during the run
  • final 27-turn incremental twist unwrapping remains explicitly unsupported
