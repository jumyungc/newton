Verification Target

This test verifies the loaded bend force-response path. Three horizontal cantilever cables receive the same downward tip force; only bend stiffness changes, so stiffer cables should deflect less and deflection times stiffness should remain nearly constant.

The analytic boundary-value case is kinematic. This case uses an actual external force, so it checks the solver path that turns load into bend.

Video Checks

The video shows the same vertical tip load applied to three cantilevers. The visible ordering is the main sanity check: low bend stiffness should sag most, high bend stiffness least. The plot below is the quantitative flat-delta-k check.

Acceptance Criteria

Passes When

  • The lowest-stiffness cable sags the most
  • The highest-stiffness cable sags the least
  • Tip markers stay ordered and close to the green 1/k guide
  • Sideways drift remains near zero

Watch For

  • A stiffer cable sags more than a softer cable
  • The cables drift sideways under a purely vertical load
  • Tip deflection keeps oscillating instead of settling

Validated Behavior

  • Bend stiffness controls loaded bend response
  • Loaded bend remains stable in the full solver
  • The bend path behaves approximately Hookean over a stiffness sweep
