Cable Twist Wiggle Buckling

Three small twist-only runs with hard-history self-contact enabled. This page is intentionally narrow: it checks whether simple contact-enabled twist cases preserve Lk = Tw + Wr.

Scope

This is a compact self-contact regression for the fixed-span twist-only cable. Contact is enabled because link conservation only has physical meaning when strand passage is prevented.

Tw is the integrated DER material-frame twist, Wr is the Klenin-Langowski discrete writhe of the centerline, and Lk error is commanded endpoint turns minus Lk. Near-zero Lk error means the imposed turns were retained as twist plus writhe.

Videos

Each case is shown with two strain measures side by side. Newton joint axes are visible in all videos.

Case 1 — Stable crown  (bend 40, twist 50, B/T 0.8, 10 turns)

B/T > 0.5: high bend resistance keeps the crown upright. Both strain measures conserve Lk = Tw + Wr throughout.

Case 2 — Soft crown  (bend 10, twist 20, B/T 0.5, 10 turns)

B/T = 0.5: softer bend compliance lets the crown droop into a loose helical coil, but no strand passage occurs. Link still conserved.

Case 3 — Snap / link loss  (bend 20, twist 50, B/T 0.4)

B/T = 0.4: twist stiffness dominates bend, driving a snap-through instability. Körner and DER strain measures compared. Three twist levels shown — same damp×3 setup for both sides at every level.

The Körner strain measure is implemented correctly per Körner & Audoly (2021): κ = 2 Im(qparent−1 qchild), bounded at ±2 by construction. The snap artifact is a property of the model — the bounded bend measure allows a snap-through path that the DER unbounded curvature binormal forecloses. This is not an implementation defect.
