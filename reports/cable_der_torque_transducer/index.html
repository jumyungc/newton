Verification Target

This report starts from material properties, not a hand-picked twist stiffness: Young's modulus E, Poisson ratio nu, cable radius r, and segment length h. For an isotropic circular rod it computes G = E / (2(1 + nu)), J = pi r^4 / 2, then gives Newton the per-joint twist stiffness k_twist = GJ / h. The material scaling check changes E, nu, r, and h, then verifies that the measured local torque response changes by the same GJ/h law. With both endpoint positions fixed and the tip rotated by theta, the analytical twist profile is phi(x) = theta x / L.

The pure-twist profile overlaps with the analytic boundary-value report. The unique purpose here is the material-property mapping: E, nu, r, and h produce k_twist = GJ / h for the cable, and changing those material inputs changes the measured torque response by the predicted amount.

Plot Checks

Material Scaling Check

This is the stiffness-mapping proof. It varies Young's modulus, radius, segment length, and Poisson ratio. For each case the report computes G = E / (2(1 + nu)), J = pi r^4 / 2, and k_twist = GJ / h, then asks the VBD angular-force kernel for torque divided by the imposed angle. The two curves should overlap; the radius case is especially diagnostic because torsional stiffness scales as r^4.

Ramp And Hold

This is the motion-time check. It verifies that the material-derived stiffness path stays well behaved while the tip is being twisted, then settles cleanly after the ramp ends.

Final Material-Mapped Torsion

At the final frame, the simulated material-frame twist should match the straight-shaft analytical profile phi(x) = theta x / L, while the centerline remains nearly straight.
